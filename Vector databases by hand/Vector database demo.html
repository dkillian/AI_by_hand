Vector database demo

A riff on Dr. Yeh’s tutorial Vector Database by Hand

1 Introduction

This is a tutorial on how vector databases work, inspired by Dr. Yeh’s Vector Database by Hand. We will explicitly follow the structure of that tutorial, but with our own data and possibly some deviations based on my own interests in what can be done with this tool.

We will proceed with establishing a corpus of text data. We will then represent that text as a set of numeric vectors. These vectors are referred to as embeddings. Next, we will store those embeddings in a vector database. Finally, we will visualize and query the vector database to demonstrate its utility.
